Installing a MEAN Stack: Step by Step


Encounter unknown terminology? Check our Glossary.

1. Create a Google Cloud Project

  1. Go to Google Cloud Home
  2. Login or create a Google Cloud Account:
    • From the top right corner, click Sign in if you have a Gmail account and enter your Gmail address and password.
    • If you do not have a Gmail account, click Get Started for Free. Follow the prompts to create your Google Cloud account.
    • After successfully signing in/creating your Google Cloud account, you will be directed back to the Google Cloud homepage where you started.
    • In the middle left of the page, notice a blue button that says “Go to console”. Click on it. This takes you to the Google Cloud Platform, where you can create a server for your web project.

2. Create a Google Cloud Server

  1. From the Google Cloud Platform, notice on the top left a three-stripe symbol. Click the three-stripe symbol to reveal a navigation menu.
  2. Scroll down the menu until you can find the option “Compute Engine”. Hover on this option to see another menu, and click “VM instances”.
  3. Notice the “Create instance” button just below the top blue “Google Cloud Platform” bar, next to “VM instances”. Click it to create an instance.
  4. Do the following:
    • Enter the instance Name (lowercase letters, no space)
    • Click on the bar below Region and choose the region you are in.
    • Click on the bar below Series to choose E2, if it is not E2.
    • Click on the bar below Machine Type, choose e2-micro or e-2 medium.
    • Scroll down the page to find Boot Disk. If it is not Ubuntu 20.04, click Change. You should see a Boot Disk menu.
    • Click the bar below the Operating System and find Ubuntu. Click Ubuntu.
    • Click the bar below Version and find Ubuntu 20.04 LTS. Click Ubuntu 20.04 LTS.
    • At the bottom, click the blue Select button. The Boot Disk menu disappears.
    • Scroll down the page to find Firewall. Check both Allow HTTP Traffic and Allow HTTPS traffic.
  5. From the bottom of the page, click Create.
  6. This may take a moment, when it is complete a bright green checkbox will appear on/near the instance.
  7. Look for the “SSH” button in the row showing the information of your newly created instance and click the downward triangle next to SSH. A little menu will pop up
  8. Click “Open in browser window”
  9. A new window will pop up and tell you it is connecting. Congratulations! You have now created and opened your own cloud server terminal command prompt! You are now ready to start installing MEAN stack components.


Note: From this point on, the lines that look like this are to be entered into the command prompt. Be patient, some commands may take time to complete, that is absolutely normal.


3. Update Your System: installing Node.js, git & dependencies

  1. Update all your system files:

    2. sudo apt update -y
    • sudo grants super user privileges which will allow the user to change system files which normally cannot be changed.
    • apt stands for Advanced Package Tool which is a command line tool for downloading and updating packages.
    • Update means to grab links to all the new files and the -y tag skips pressing yes for your convenience.


    Note:     This does not actually install your files, it just downloads them.

  2. Upgrade all your system files:

    3. sudo apt upgrade -y
    • Apt upgrade now installs all of the files we downloaded in the last step


4. Install MongoDB and start services

  1. Install dependencies necessary to add a new repository over HTTPS:

    2. sudo apt install -y dirmngr gnupg apt-transport-https ca-certificates software-properties-common
  2. Import MongoDB’s repository GPG key:

    3. wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -


    Note:     At the time of this writing, the latest version of MongoDB is 4.4. To install a different version of MongoDB, just replace 4.4 with your preferred version number. Go to Mongo Release Notes to see the latest version of MongoDB.

  3. Add the MongoDB repository:

    4. sudo add-apt-repository 'deb [arch=amd64] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse'
  4. Once the repository is enabled, install the mongoDB package:

    5. sudo apt install -y mongodb-org
    • These packages include the Mongo daemon, init scripts, configurations, interactive JavaScript shell, as well as various MongoDB tools and utilities.
  5. Start the MongoDB daemon and enable it to start on boot:

    6. sudo systemctl enable --now mongod

5. Install Node and other dependencies

  1. Use the official node source script to get the latest version of node.js:

    2. curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
    • Curl is a tool preferred for automation, since it is designed to work without user interaction. curl can also transfer multiple files at once.


    Note:     This hopefully lets us know some various methods of installing some of the stuff we have now downloaded “sudo apt-get install -y nodejs” is a good example similar to what we will install below.

  2. Install node.js, git and other dependencies:

    3. sudo apt-get install -y nodejs gcc g++ make git
    • Using apt we are now going to install the nodejs package, gcc, g++, make and git.


    Note:     This is a combination of the commands in the comments when we downloaded node. Further note: gcc and g++ are open source versions of c and c++ respectively, Make is a compiler and git is Git.

  3. Clone the MEAN stack repository:

    4. sudo git clone https://github.com/meanjs/mean
    • Using git we are going to clone the MeanJS repo.
  4. Check which version node.js is installed:

    5. node -v


    Note:     As of this writing the version is v14.15.1

6. Install Yarn, Gulp, and SASS

  1. We need to change permissions to allow us to install npm and packages.

    2. mkdir ~/.npm-global

    npm config set prefix '~/.npm-global'

    echo "export PATH=~/.npm-global/bin:$PATH" >> ~/.profile

    source ~/.profile
  2. After installing node.js, we can now install Yarn and Gulp. Type this to go to the MEAN stack directory, which you have cloned earlier with git:

    3. cd mean
  3. Next, we will install Yarn SASS and Gulp using NPM. To install Yarn, use this command:

    4. npm install -g yarn
  4. Install node-sass with this command:

    5. npm install -g node-sass
  5. Install Gulp with this command:

    6. sudo npm install --global gulp-cli
  6. To update all the files you installed, use this command:

    7. npm update
  7. Now install all of the updated files:

    8. npm install

    npm audit fix

This completes the setup of the MEAN Stack. Continue to the next tutorial by clicking the link below.
